# 功能更新方案 v7 —— v2.3.0(功能移除)

用户反馈两项功能实际无法使用,予以删除。离线翻译、拍照在线识别保留。

## ① 删除离线图片识别(ML Kit Text Recognition)
- 删 `mlkit_ocr_service.dart`;pubspec 去 `google_mlkit_text_recognition`。
- `settings_service`:删 `OcrMode` 枚举、`ocrMode` 字段与 `cfg_ocr_mode` 持久化。
- `home_page._runOcr`:去掉 ocrMode 分支,恢复为**仅在线视觉模型**(需配置 API);
  删 `_mlkitOcr` 字段、dispose、import。
- `settings_page`:删"文字识别(OCR)方式"单选区(保留"优先离线翻译"开关,归到
  "翻译方式"小节)。
- **保留**:拍照/相册导入 → 在线视觉大模型识别(gpt-4o / qwen-vl 等);离线翻译。

## ② 删除音色选择
系统 TTS 音色/男女声/音调/"下载更多声音"实际不可用,整体移除,TTS 回落系统默认声音。
- `tts_service`:删 `VoiceInfo`、`voiceLanguage/voiceName/pitch`、`availableVoices/Languages`、
  `loadVoices/applyVoiceSettings/applyPreset/_pickVoiceByGender/voicesForLanguage/
  _femaleHints/_maleHints/speakPreview/hasVoiceFor/openInstallVoiceData/_tryLaunch`;
  去 `android_intent_plus`、`settings_service` import;play/playFrom/synthToFile 里
  不再调 applyVoiceSettings;init 不再 setPitch/loadVoices。**保留**核心朗读、听写切词、
  重复间隔、synthToFile 导出。
- `settings_service`:删 `VoicePreset` 枚举、`voicePreset/voiceLanguage/voiceName/pitch`
  字段与对应持久化键。
- `settings_page`:删整个"音色选择"UI 段与 `_applyPreset/_changeLanguage/_changeVoice/
  _changePitch/_preview/_installVoice/_langLabel/_buildLangItems` 及
  `_languages/_voicesLoading/_previewing` 状态。
- `reader_page`:删音色底部面板 `_showVoiceSheet` 与 `_applyPresetInReader/
  _changePitchInReader`;去 init 里的 voice 灌参。**app-bar 原"音色"按钮改为直接打开
  "已生成音频"列表**(原先该入口藏在音色面板里,避免功能丢失)。
- `pubspec.yaml`:删 `android_intent_plus`。
- `AndroidManifest.xml`:`<queries>` 回退为仅 TTS_SERVICE + IMAGE_CAPTURE
  (删 INSTALL_TTS_DATA / TTS_SETTINGS / market / https)。

## 保留未动
- 听写每词重复间隔(v2.2.0 ①)、离线翻译 + 在线回退(v2.2.0 ③翻译半)。
